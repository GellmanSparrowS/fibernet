FiberNet

Computational Design of Fiber Network Metamaterials

Overview

FiberNet is a Python toolkit for computational design of fiber network metamaterials. It provides a closed-loop workflow from parametric structure generation through GPU-accelerated simulation to machine learning and reinforcement learning optimization.

Generation → Simulation → Feature Extraction → Machine Learning → Reinforcement Learning

Installation

# Core installation
pip install fibernet

# Full installation (ML + RL + visualization + simulation)
pip install fibernet[full]

Key Features

  • Structure Generation — 12+ 2D and 15+ 3D unit types with parametric control
  • GPU Simulation — Taichi-based mass-spring dynamics with automatic differentiation
  • Feature Extraction — 94-dim (2D) and 60-dim (3D) feature vectors
  • Machine Learning — Regression, classification, cross-validation pipelines
  • Reinforcement Learning — Gymnasium-compatible environments with CEM and Bayesian optimization
  • Visualization — 2D/3D rendering with matplotlib and PyVista

Quick Start

# Generate a 2D honeycomb structure
import fibernet as fn
g = fn.pattern_2d(unit="honeycomb", grid=(3, 3), n_pts_per_side=5)

# Run simulation
result = fn.TaichiEngine().stretch_test(g, target_stretch=1.5)

# Extract features for ML
features = fn.GraphFeatureExtractor().extract(g)
